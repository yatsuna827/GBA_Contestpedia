ID 効果
A00 （追加効果なし）
B00 順番が1匹目なら6点、それ以外なら2点。

B02 行動順に応じて得点が変動。（早いほうから順に1点、2点、4点、6点）

B08 自分より前のポケモンたちの得点合計の半分（切り捨て）+1点の得点。ただしマイナスの得点は無視する。
B09 調子ボーナスを3倍で計算する。
C00 そのターン中、1回だけ妨害を受けない。
C01 そのターン中、妨害を受けない。
C02 調子が上がる。
D00 直前のポケモンに4点の減点。
D01 直前のポケモンに3点の減点。
D02 自分より前のポケモンたちに3点の減点。
D03 自分より前のポケモンたちに2点の減点。
D04 得点が1点以上あれば得点の半分（切り上げ）の減点、0以下の場合は1点の減点。
D05 同じタイプのアピールをしていたら4点、そうでないなら1点の減点。
D06 コンボ待機状態なら5点、そうでないなら1点減点。
D07 自分より前のポケモンたちのコンボ待機状態を解除する。
D08 自分より前のポケモンたちの調子を0に戻す。
D09 自分より後のポケモンたちを緊張させる。
E00 自分の次のターンの行動順を1番目に予約する。
E01 自分の次のターンの行動順を4番目に予約する。
E02 全員の次のターンの行動順がランダムに予約される。
F00 そのターン中、 妨害による減点が2倍になる。
F01 自分より前のポケモンたちに4点の減点。自分は次のターン、行動不能状態になる。
F02 自分は次ターン以降、 行動不能状態になる。
G00 そのターン中、エキサイトが変動しなくなる。
H00 部門によらずエキサイトが上昇する。
I00 連続で使ってもペナルティがつかない。
